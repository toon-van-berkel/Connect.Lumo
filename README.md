# LUMO Connect

LUMO Connect is ontwikkeld als ondersteunend onderdeel van het LUMO-project. Deze repository bevat de code die nodig is om de communicatie en connectiviteit binnen het systeem mogelijk te maken.

Waar LUMO zich richt op het toegankelijker maken van hardlopen voor mensen met een auditieve beperking, zorgt LUMO Connect voor de technische verbinding achter deze ervaring.

## Wat doet LUMO Connect?

LUMO Connect ondersteunt de werking van het LUMO-systeem door:
- signalen binnen het systeem te verwerken
- communicatie tussen onderdelen mogelijk te maken
- de koppeling tussen feedback en gebruiker te ondersteunen

Hierdoor kan LUMO belangrijke momenten, zoals een startsignaal, een passerende loper of het behalen van de finish, op een betrouwbare manier doorgeven via visuele en voelbare feedback.

## Relatie met LUMO

Deze repository is onderdeel van het bredere LUMO-project. LUMO is ontworpen om hardlopen toegankelijker, veiliger en inclusiever te maken voor mensen met een auditieve beperking. LUMO Connect vormt hierbij de technische basis die deze functionaliteiten ondersteunt.

## Doel van deze repository

Het doel van LUMO Connect is om een stabiele en functionele technische verbinding te bieden binnen het LUMO-systeem, zodat gebruikers op een duidelijke en betrouwbare manier feedback kunnen ontvangen tijdens het hardlopen.
